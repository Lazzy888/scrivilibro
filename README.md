## Descrizione

**ScriviLibro** è una Progressive Web App progettata per chi vuole scrivere libri,
racconti o qualsiasi testo lungo, utilizzando sia la tastiera che la dettatura vocale.
Integra strumenti di intelligenza artificiale (via Groq API) per riscrivere,
espandere, riassumere e migliorare il testo.

### Funzionalità principali

- 🎙️ **Dettatura vocale** in italiano in tempo reale
- 📚 **Gestione capitoli** — crea, rinomina, riordina, elimina
- 📁 **Multi-libro** — salva e ricarica più libri indipendenti
- 🤖 **Strumenti AI** — riscrittura, stile, tono, espansione, riassunto…
- ✂️ **Editor avanzato** — frasi, testo diretto, cerca & sostituisci
- 📤 **Esportazione** — TXT e DOCX (Word / LibreOffice / Google Docs)
- 📴 **Offline parziale** — Service Worker per uso senza connessione
- 📱 **Installabile** come app nativa su Android, iPhone, desktop

---

## Uso

Apri `index.html` in un browser moderno oppure distribuisci via web server HTTPS
per sfruttare le funzionalità PWA complete (Service Worker, installazione).
